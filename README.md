An OpenClaw-powered agentic framework for building digital sanctuaries. Built on BlockSmith's "Zero Material" philosophy, this project provides a vaultless, intent-first operating system for AI agents.

## 🏛️ The Philosophy

OpenBot isn't just a chatbot; it's a **Sovereign Agent**. We follow the **Ontological Security Doctrine (OSD)**:
- **Zero Material:** We never store what can be derived. Logic is ephemeral; secrets are non-existent.
- **Intent-First (IRP):** Architecture grows from declared `TELOS`. Code is an auditable reflection of intent.
- **Asymmetric Friction:** Zero friction for the human; soul-crushing complexity for the attacker.
- **Engineered Kindness:** Security as an act of care, respecting human dignity as an original work of art.

## 🚀 Key Features

- **Vaultless Authentication:** Eliminates static secrets and central liability.
- **Persistent Evolution:** Automated self-auditing and skill improvement via `EVOLUTION.md`.
- **Architectural Rigor:** Standardized `SOUL.md`, `CONSTITUTION.md`, and `STYLE.md` mandates.
- **Deterministic Derivation:** Powered by the Pleione cybersecurity suite principles.
