# TOOL: trash file (trashit)

## INTENT
Move a file to a dedicated trash directory (`~/.trash/`) to clear the workspace while keeping it accessible for recovery.

## STRATEGY (The "Waste Management" Phase)
1.  **Define Target**: Set `TRASH_DIR` to `/home/michelc/.trash/` (create it if missing).
2.  **Move & Version**: Run `mv <file> ${TRASH_DIR}/<file>_$(date +%s)`.
3.  **Cleanup Verification**: Run `ls <file>` to confirm it's no longer in the active path.

## MANDATE
- **Temporary Holding**: Trash is for things that are no longer part of the current `TELOS` but might be needed for retrospective analysis.
- **Privacy Check**: Ensure no PII is trashed in an unencrypted state if possible (though `~/.trash` is locally private).
