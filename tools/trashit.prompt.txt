# TOOL: trash file (trashit)

## INTENT
Move a file to the `.trash` sanctuary using the `bin/trashit.sh` script to ensure auditable, reversible de-materialization.

## STRATEGY (The "Waste Management" Phase)
1.  **Execute Protocol**: Run `./bin/trashit.sh <file>`.
2.  **RCS Awareness**: The script will automatically check in any RCS-tracked files before trashing.
3.  **Conflict Resolution**: The script handles name collisions using the `*~n` versioning format.
4.  **Verification**: Confirm the file is no longer in the active workspace path.

## MANDATE
- **Zero Destruction**: Never use `rm`. Always use `trashit.sh`.
- **Temporal Clarity**: The `.trash` folder is for items that are no longer part of the current `TELOS` but retain historical value.
